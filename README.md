# Projet Aegis Bank

Ce projet présente une landing page responsive pour Aegis Bank, une banque mobile et écoresponsable, créée pour un projet Q1-Q3.

## Fonctionnalités implémentées
- Structure HTML sémantique avec balises ARIA
- Convention BEM appliquée
- Système de design avec variables CSS
- Animations CSS pures (fadeIn, pulse, hover transitions)
- Sections RGPD (mentions légales, politique confidentialité)
- Responsive mobile-first
- Aucun cookie utilisé

## Technologies
- HTML5 sémantique
- CSS3 avec animations
- Mobile-first responsive

## Validation
- W3C HTML/CSS validé
- WCAG 2.1 AA compliant
- Performance Lighthouse ≥ 90 (mobile)

## Déploiement
- GitHub: https://github.com/monatsk/aegis-bank-
